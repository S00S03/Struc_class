\subsubsection*{template$<$typename T$>$\newline
class Stack$<$ T $>$}
Clase plantilla que implementa una pila (\doxylink{classStack}{Stack}) dinámica. 

Implementa una estructura LIFO (Last In, First Out). La memoria se gestiona dinámicamente y se redimensiona automáticamente cuando se alcanza la capacidad máxima.
